Lunar Tab is a New Tab Page extension for Google Chrome. It displays the phase of the moon, moonrise, moonset, and some astrological information for the moon in its current position. I wanted the design to look flat, cute, and somewhat minimalistic.

Technologies used:

  • HTML
  • CSS
  • JavaScript
  • Google Geocoding API
