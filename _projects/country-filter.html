Country Filter is a userscript for eBay that hides items being sold from certain countries. It allows you to make a list of countries you want to exclude from your search results and then hides them, making it easier to look at the remaining listings.

This userscript was made because eBay's default settings for international listings are incredibly limited. One can only restrict searches to North America or worldwide.

Disclaimer: Country Filter is not affiliated with eBay in any way whatsoever.

Technologies used:

  • JavaScript
  • jQuery
  • HTML
  • CSS
